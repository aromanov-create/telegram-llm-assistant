# Changelog

В этом файле фиксируются значимые изменения проекта.

Формат близок к Keep a Changelog, но адаптирован под небольшой инженерный проект.

## [Unreleased]

### Added
- Добавлен `CHANGELOG.md` для ведения значимых изменений проекта.
- Добавлен `AGENTS.md` с правилами для AI-агентов по коммитам, документации и обязательному обновлению changelog.

### Правило ведения
- Любой значимый кодовый, конфигурационный или документационный коммит должен сопровождаться обновлением этого файла.
- Если изменение не влияет на пользователей, разработчиков, конфигурацию, запуск или сопровождение проекта, его можно не добавлять.

## [0.1.0] - 2026-03-20

### Added
- Добавлен Telegram-бот, который принимает текст, голосовые сообщения, фото и видео.
- Добавлена расшифровка голосовых сообщений через Whisper.
- Добавлен запуск задач через `claude` CLI.
- Добавлены `systemd` unit-файлы для бота и SSH-прокси.
- Добавлен `assistant-bot.env.example` с шаблоном конфигурации.
- Добавлен `LICENSE`.
- Добавлен `README.md` с описанием проекта, сценариями использования и инструкцией по запуску.

### Changed
- Настройка прокси вынесена из `assistant-bot.service` в `assistant-bot.env`.
- README дополнен практическими кейсами использования: почта, API, задачи, документы, локальная автоматизация.
- Улучшена типизация `bot.py` и добавлены поясняющие комментарии в неочевидных местах.

### Fixed
- Исправлены проблемы со стартом и зависанием бота при нестабильной сети.
- Добавлен watchdog для перезапуска процесса через `systemd`, если Telegram API недоступен слишком долго.
- Устранены типовые ошибки `Optional` и прочие замечания анализатора в `bot.py`.

### Security
- Секреты вынесены из кода в env-файл.
- История репозитория очищена от токена, логов и личных заметок.
- Из публичной версии проекта удалены приватные рабочие артефакты.
